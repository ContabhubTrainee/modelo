# Pasta de Imagens

Coloque suas logos aqui:

## Logos necessárias:

1. **logo-aura8.png** - Logo para tema claro (fundo claro)
2. **logo-aura8-branca.png** - Logo para tema escuro (fundo escuro)

## Especificações recomendadas:

- Formato: PNG com fundo transparente
- Tamanho recomendado: 280x64px (ou proporção similar)
- Resolução: Mínimo 2x para telas de alta densidade (retina)

## Como adicionar:

1. Coloque o arquivo `logo-aura8.png` (logo para tema claro) nesta pasta
2. Coloque o arquivo `logo-aura8-branca.png` (logo para tema escuro) nesta pasta
3. As logos serão carregadas automaticamente conforme o tema selecionado

## Nota:

Se você tiver apenas uma logo, pode usar a mesma para ambos os temas, mas é recomendado ter versões diferentes para melhor contraste e legibilidade.
